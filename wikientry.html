WikiEntry

WikiEntry allows users to read and edit wikis without using a web client. This can save user's bandwitch, which is very important for users with limited connection

MAUI version

The original version, no longer in development. It will be replaced by Avalonia-based version in the future

Few limitations

  • Only for Android and Windows sideloading (no iOS and macOS support yet)
  • No Linux support (yet, should get fixed with Avalonia)
  • Lack of editing toolbar
  • Lack of "Show changes" functionality
  • Only in English
  • The performance may be bit poorer

Android

Instructions

  1. Download the file to your computer
  2. Use AAB to APK converter to make it launchable on your phone
  3. Copy the file to your phone
  4. Run the executable
  5. Hopefully, it should install
  • Android download (aab)

Windows

Instructions

  1. Download the file
  2. Unzip the file
  3. Run the MSIX executable
  4. It should install and run automatically
  • Windows download (msix)
